## 快捷键
图标含义

- `⌘` = `Command`
- `⌥` = `Option`/`Alt`
- `⌃` = `Control`
- `⇧` = `Shift`
- `↩` = `Enter`

## 常用操作

- 快速构建运行：`⌘+R`
- 构建`⌘+B`
- 清除缓存`⌘+⇧+K`
- 展开/关闭左侧导航 `⌘+0`
- 展开/关闭右侧导航 `⌘+⇧+0`
- 定位文件`⌘+⇧+J`
- 后退`⌘+[`
- 前进`⌘+]`
- 显示/关闭控制台`⌘+⇧+Y`

- 打开设置 `⌘+,`
- 隐藏编辑器 `⌘+H`
- 退出编辑器 `⌘+Q`
- 显示/隐藏头部 `⌘+⌥+T`
- 新建标签组 `⌘+T`
- 新建左右标签组`⌘+⌃+T`
- 复制Xcode编辑器窗口 `⌘+⇧+T`
- 新建项目`⌘+⇧+T`
- 新建文件 `⌘+N`
- 新建文件夹 `⌘+⇧+N`
- 打开本地文件 `⌘+O`
- 生成当前文件所有定义（Generated Interface
- 收起/展开当前代码块 `⌘+⌥+←`/`⌘+⌥+→`

### 基础导航
+ `⌘ + 0-9`: 显示/隐藏不同面板
+ `⌘ + 0`: 显示/隐藏导航器面板
+ `⌘ + 1`: 项目导航器
+ `⌘ + 2`: 源代码控制导航器
+ `⌘ + 3`: 符号导航器
+ `⌘ + 4`: 查找导航器

### 编辑代码
+ `⌘ + [`: 向左缩进
+ `⌘ + ]`: 向右缩进
+ `⌘ + /`: 注释/取消注释
+ `⌘ + L`: 跳转到指定行

### 代码导航
+ `⌘ + 点击`: 跳转到定义
+ `⌘ + ⇧ + O`: 打开使用手册
+ `⌃ + 6`: 显示当前文件的方法列表

### 搜索
+ `⌘ + F`: 在当前文件中查找
+ `⌘ + ⇧ + F`: 在项目中查找
+ `⌘ + E`: 使用选中的文本进行搜索（需要先执行`⌘ + F`或打开`查找`功能）

### 预览
+ `⌘ + ⌥ + **↩**`: 快速浏览/关闭预览窗口
+ `⌘ + ⌥ + P`: 刷新预览窗口

### 编译和运行
+ `⌘ + B`: 构建
+ `⌘ + R`: 运行
+ `⌘ + .`: 停止运行
+ `⌘ + U`: 运行测试
+ `⌘ + ⇧ + K`: 清理构建项目

### 调试
+ `⌘ + \`: 设置/取消断点
+ `⌘ + Y`: 启用/禁用所有断点
+ `F6`: 单步执行
+ `F7`: 步入
+ `F8`: 步过

### 代码补全和提示
+ `⌃ + Space`: 代码补全
+ `⌥ + 点击`: 显示文档弹窗
+ `⌘ + ⌥ + /`: 添加文档注释

### 窗口管理
+ `⌘ + 0`: 显示/隐藏导航器面板（左侧）
+ `⌘ + ⇧ + Y`: 显示/隐藏调试区域（底部）
+ `⌘ + ⌥ + 0`: 显示/隐藏工具区域（右侧）
+ `⌘ + ⌥ + ⌃ + ⌘`: 显示/隐藏模拟器

### 实用功能
+ `⌘ + ⇧ + K`: 清理项目
+ `⌘ + ,`: 打开偏好设置
+ `⌘ + ⌥ + ⌃ + G`: 重新生成索引

## 属性图标含义
![](/images/notes/2026/01/2026-01-11-12.png)

+ `K` (绿色) - 关键字 (Keyword)
    - 如 `some`, `none` 等 Swift 关键字
+ `I` (蓝色) - 实例/初始化器 (Instance/Initializer)
    - 实例方法
    - 初始化方法 `init`
+ `S` (紫色) - 结构体 (Structure)
    - 结构体定义
+ `M` (蓝色) - 方法 (Method)
    - 类或结构体中的方法
+ `T` (橙色) - 类型 (Type)
    - 类、枚举等类型定义
+ `P` (紫色) - 协议 (Protocol)
    - 协议定义
+ `V` (绿色) - 变量/属性 (Variable/Property)
    - 实例变量
    - 属性
+ `E` (红色) - 枚举 (`Enum`)
    - 枚举定义及其成员
+ `C` (蓝色) - 类 (Class)
    - 类定义
