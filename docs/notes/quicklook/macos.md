### 方法一：快捷键 (最快、最推荐)

1.  打开访达（Finder）。
2.  按下快捷键组合：**`Command` + `Shift` + `.`**。

会立刻看到隐藏文件和文件夹（它们的图标会是半透明的）。再次按下这套快捷键，就可以将它们重新隐藏。

**临时切换，适合日常使用。**

---

### 方法二：使用终端命令 (永久显示)

1.  **打开“终端” (Terminal)**
    *   `应用程序` > `实用工具`。
    *   或者直接按 `Command` + `空格` 启动聚焦搜索，输入“终端”并回车。

2.  **输入显示命令**
    ```bash
    defaults write com.apple.finder AppleShowAllFiles -bool true
    ```

3.  **重启访达使设置生效**
    ```bash
    killall Finder
    ```

**恢复隐藏：**
将命令中的 `true` 改为 `false` 即可：
```bash
defaults write com.apple.finder AppleShowAllFiles -bool false
killall Finder
```

---

### 方法三：在访达的“前往文件夹”中查看

如果只是想访问某个特定的隐藏文件夹（比如 `/usr` 或 `/private`），但不想显示所有隐藏文件，可以使用这个方法。

1.  打开访达。
2.  点击屏幕顶部的菜单栏中的 **前往 (Go)**。
3.  选择 **前往文件夹... (Go to Folder...)**，或者直接使用快捷键 **`Command` + `Shift` + `G`**。
4.  在弹出的对话框中，输入想访问的隐藏文件夹的路径（例如 `/private/var`），然后点击“前往”。

---

### 为什么会有隐藏文件？

macOS 会隐藏一些对系统正常运行至关重要的文件和文件夹，目的是**防止用户误删或误改导致系统出问题**。这些文件通常以 `.` 开头（如 `.DS_Store`, `.zshrc`），或者被系统设置了特殊的隐藏属性。

**⚠️ 警告**：在处理隐藏文件时请务必小心，不要删除或修改任何你不确切了解其用途的文件。

### 总结

| 方法        | 操作方式                                    | 效果               | 优点/缺点                      |
| :-------- | :-------------------------------------- | :--------------- | :------------------------- |
| **键盘快捷键** | `Command` + `Shift` + `.`               | 临时切换显示/隐藏        | **推荐**：快速、方便、安全，不改变系统默认设置。 |
| **终端命令**  | `defaults write ...` + `killall Finder` | 永久更改显示状态（直到再次更改） | 适合需要一直查看隐藏文件的开发者，但操作稍复杂。   |
| **前往文件夹** | `Command` + `Shift` + `G`               | 访问特定的隐藏路径        | 精准，不干扰全局设置，但无法浏览所有隐藏文件。    |
